Getting started with vim

If you find vim confusing, hard to use, and difficult for editing text, don't worry because we got you covered. So today, we're going to talk about simple vim commands for begginers.

The basic commands that you need to know right now



:w FILE.txt Means to save the file

:q to quit the program

:r to redo the changes

:u tp undo the changes

How to make vim look appealing?

Do you want to make vim look like an appealing text editor? Yes, you can do that in vim right away by entering this commands.



:colorscheme evening

:set signcolumn=yes

:set nu

Now your text editor looks appealing and neat, but reloading im vim means that you have to set it up all over again. Don't worry about it, you can save those changes permanently in vim by creating a .vimrc file where you can save the setting by typing the right syntax in .vimrc file to load them automatically when you start vim.
