Basics of Vim

  • :w

    • Save any file that you want to save

  • i

    • Inserting any text or code that you want to enter

  • ESC key

    • This will enable you to only view the file and enter command mode

  • :q

    • To quit the program after you save it

These will help you to get starting with Vim. Of course, there's plenty of more keybindings and even some specific ones that you may not use them that frequently, but you can search for help by opening Vim and typing in the command :help. This will help your find the specific keybindings or other features that Vim offers. (Keep in mind that not everything that you want to find is there so it would be better to try search the web for other specific information)
